How Accreditation Works

THE ACCREDITATION PIPELINE

Every request travels through
4 approval stages before a pass is issued

The Process
01
edit_note
Submit a Request
A Requestor fills in the applicant's name, passport, nationality, role and event — then submits into the pipeline.
east
02
account_tree
Pipeline Review
The request passes through up to 4 sequential stages. Each authority can approve, reject, or request more info.
east
03
verified
Pass Issued
Once all stages clear, a QR-coded accreditation pass is generated and delivered to the applicant.
Pipeline Stages in Detail
business
Stage 1 — FA Owner
Federation / Association Owner
First Review

Verifies the applicant's role and federation affiliation. If anything is unclear, they can pause the pipeline and request more documents from the requestor.

check Approve → Stage 2 close Reject → Closed help_outline Request Info → Paused
location_on
Stage 2 — Zone Owner
Venue / Zone Access Authority
Access Check

Assigns the specific physical zones the applicant can enter — Field, Media Tribune, VIP Lounge, Back of House, etc. Reviews and confirms appropriate access level.

check Approve + zones → Stage 3 close Reject → Closed help_outline Request Info → Paused
photo_camera
Stage 3 — Media Owner
Media & Communications Authority
Media Check

Validates press credentials, broadcaster affiliations, and production documentation. Required for any role with media, camera, or broadcast access to the venue.

check Approve → Stage 4 close Reject → Closed help_outline Request Info → Paused
security
Stage 4 — MOI
Ministry of Interior, Qatar
MOI Required

Mandatory security clearance for VIPs, foreign nationals, and all MOI-flagged events. The Ministry of Interior performs a background check before the final pass is issued.

check Cleared → Pass issued close Denied → Closed
Request States
In Review
Request is actively being reviewed at a pipeline stage.
Info Requested
Reviewer needs more documents. Pipeline is paused.
Approved
All stages cleared. Pass generated and sent.
Rejected
Denied at a stage. A new request must be submitted.
Roles & Responsibilities
person_add
Requestor
  • Submits requests on behalf of applicants
  • Monitors pipeline progress in real time
  • Responds to info requests & resubmits
  • Uses bulk upload for large groups
admin_panel_settings
FA Owner
  • First reviewer in the pipeline
  • Verifies role and federation affiliation
  • Manages events and accreditation windows
location_on
Zone Owner
  • Controls venue access zones
  • Assigns zone permissions to each pass
  • Reviews zone access requirements
photo_camera
Media Owner
  • Verifies press and broadcast credentials
  • Approves camera and equipment access
security
MOI
  • Background & security clearance
  • Required for VIPs and foreign nationals
  • Final authority on MOI-flagged events
id_card
Accredited User
  • Receives the approved pass by email
  • Scans QR code at entry checkpoints
Tips
task_alt
Submit early — Pipeline review can take 2–3 business days per stage. Submit well before the event date.
upload_file
Use bulk upload for groups — Download the CSV template, fill in all applicants, and upload in one go.
help_outline
Respond to info requests promptly — The pipeline pauses until you resubmit. Check notifications regularly.
qr_code_2
Save your QR pass — Download or add to mobile wallet. The QR code is required at every entry checkpoint.
Questions? Contact accreditation@qoc.qa
